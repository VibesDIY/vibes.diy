# Code of Conduct

This project follows the Contributor Covenant Code of Conduct (version 2.1).

- Be respectful, constructive, and professional.
- Harassment, discrimination, hate speech, doxxing, or personal attacks are not acceptable.
- If someone asks you to stop harmful behavior, stop.

## Scope

This Code of Conduct applies in all project spaces, including issues, pull requests, discussions, and other community channels where you represent this project.

## Reporting

Report unacceptable behavior to **help@vibes.diy**.

All reports will be reviewed by maintainers and handled as confidentially as possible.

## Enforcement

Maintainers may remove content, limit interactions, or restrict participation for behavior that violates this Code of Conduct.
