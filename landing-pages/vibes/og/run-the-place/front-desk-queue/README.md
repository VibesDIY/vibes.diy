# front-desk-queue

> Walk-in and reservation queue for salons, Pilates studios, and dental offices. Today as a timeline; tap to add a name + phone + service + duration; check in or no-show buttons; "running late" auto-highlight after the slot start time passes. Drag to reschedule. useFireproof for live sync between front-desk and stylist views. Single-file React. Tone: clean booking-page feel, mobile-first.

Live at [https://vibes.diy/vibe/og/front-desk-queue](https://vibes.diy/vibe/og/front-desk-queue)

Single-file React app built with [vibes.diy](https://vibes.diy). Visit the live url to manage access.

## Run it

```sh
npx vibes-diy push     # uploads App.jsx, prints a live HTTPS URL
```

Edit [App.jsx](App.jsx) and push again to iterate.

## Commands

- `npx vibes-diy push` — deploy the current directory
- `npx vibes-diy push --instant-join` — deploy with auto-accept sharing
- `npx vibes-diy generate "prompt"` — generate a new app from a prompt
- `npx vibes-diy help` — full command list
