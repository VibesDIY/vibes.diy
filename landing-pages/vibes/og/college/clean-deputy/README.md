# clean-deputy

> Build a funny anonymous community app called Clean Deputy where people can quietly report annoying shared-space situations like abandoned laundry, disgusting fridges, mystery spills, moldy leftovers, unwashed dishes, destroyed bathrooms, or weird hallway chaos without directly confronting anyone. Users should upload photos and short descriptions, and the apps AI should generate playful roast commentary about the scene. The tone should feel like a mixture of neighborhood watch, hall monitor satire, and group-chat humor. The goal is not cruelty but social pressure through comedy. The app should include a public leaderboard showing which dorm floors, apartments, kitchens, or anonymous offenders got roasted the most. Users should react, vote, add jokes, and escalate ongoing legendary mess situations into local campus lore. The visual design should feel like a fake municipal authority mixed with internet meme culture. Everyone should feel like they are participating in a ridiculous shared civilization project to maintain order in chaotic communal living spaces. $STYLE

Live at [https://vibes.diy/vibe/jchris/clean-deputy](https://vibes.diy/vibe/jchris/clean-deputy)

Single-file React app built with [vibes.diy](https://vibes.diy). Visit the live url to manage access.

## Run it

```sh
npx vibes-diy push     # uploads App.jsx, prints a live HTTPS URL
```

Edit [App.jsx](App.jsx) and push again to iterate.

## Commands

- `npx vibes-diy push` — deploy the current directory
- `npx vibes-diy push --instant-join` — deploy with auto-accept sharing
- `npx vibes-diy generate "prompt"` — generate a new app from a prompt
- `npx vibes-diy help` — full command list
