# study-roulette

> Build a social productivity app called Study Roulette where students can mark themselves as ready to study and get randomly paired or grouped with other nearby students for short timed work sessions. The app should feel spontaneous, social, and slightly chaotic in a good way, like speed dating for productivity. It should support quick pairings, temporary study circles, themed sessions, accountability bursts, and low-pressure social studying. Some people may secretly hope to meet someone interesting, while others genuinely just need motivation to focus, and the app should comfortably support both energies without becoming a dating app. The atmosphere should feel alive and communal, like late-night library culture during finals week. Include visual indicators that show who is currently active, what subjects people are working on, and what kind of energy they want, like silent focus mode versus collaborative chaos mode. The app should celebrate streaks, accidental friendships, and the strange temporary intimacy of studying beside strangers. $STYLE

Live at [https://vibes.diy/vibe/jchris/study-roulette](https://vibes.diy/vibe/jchris/study-roulette)

Single-file React app built with [vibes.diy](https://vibes.diy). Visit the live url to manage access.

## Run it

```sh
npx vibes-diy push     # uploads App.jsx, prints a live HTTPS URL
```

Edit [App.jsx](App.jsx) and push again to iterate.

## Commands

- `npx vibes-diy push` — deploy the current directory
- `npx vibes-diy push --instant-join` — deploy with auto-accept sharing
- `npx vibes-diy generate "prompt"` — generate a new app from a prompt
- `npx vibes-diy help` — full command list
