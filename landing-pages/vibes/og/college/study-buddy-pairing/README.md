# study-buddy-pairing

> Random study pairings app. Mark yourself ready; the app pairs or groups you with nearby students for short timed focus sessions. Show who's active, what they're studying, and their energy mode (silent focus vs collaborative chaos). Celebrate streaks and accidental study friendships. Save sessions in Fireproof.

Live at [https://vibes.diy/vibe/og/study-buddy-pairing](https://vibes.diy/vibe/og/study-buddy-pairing)

Single-file React app built with [vibes.diy](https://vibes.diy). Visit the live url to manage access.

## Run it

```sh
npx vibes-diy push     # uploads App.jsx, prints a live HTTPS URL
```

Edit [App.jsx](App.jsx) and push again to iterate.

## Commands

- `npx vibes-diy push` — deploy the current directory
- `npx vibes-diy push --instant-join` — deploy with auto-accept sharing
- `npx vibes-diy generate "prompt"` — generate a new app from a prompt
- `npx vibes-diy help` — full command list
