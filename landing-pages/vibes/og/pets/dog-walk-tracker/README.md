# dog-walk-tracker

> Dog walk coordinator for a family or roommates. List of dogs. Tap to start a walk timer; on stop, log walker name, duration, and an optional note (energy level, poop count, anything noticed). Daily summary shows total walk time per dog and who walked them. Streak counter for consecutive days walked. Fireproof for sync across devices. No images.

Live at [https://vibes.diy/vibe/jchris/dog-walk-tracker](https://vibes.diy/vibe/jchris/dog-walk-tracker)

Single-file React app built with [vibes.diy](https://vibes.diy). Visit the live url to manage access.

## Run it

```sh
npx vibes-diy push     # uploads App.jsx, prints a live HTTPS URL
```

Edit [App.jsx](App.jsx) and push again to iterate.

## Commands

- `npx vibes-diy push` — deploy the current directory
- `npx vibes-diy push --instant-join` — deploy with auto-accept sharing
- `npx vibes-diy generate "prompt"` — generate a new app from a prompt
- `npx vibes-diy help` — full command list
