# arena-subsidy-scoreboard

> Build an interactive data visualization called The Subsidy Scoreboard. It ranks how 16 recent major North American sports-arena projects were funded, by the share of construction cost paid with PRIVATE capital. The whole point of the page is one clean, honest, animated bar chart that lets a reader see at a glance where the Portland Moda Center proposal sits next to its peers. On first load, seed a Fireproof database named arena-deals with one document per project, using the project slug as the _id so seeding is idempotent and never duplicates; then render everything from a live query of that database. Each document has fields: name, location, year, type (renovation or new build), costM (total cost in millions of dollars), publicM, privateM, privatePct (private share, an integer 0 to 100), rent (a short note on annual public rent), terms (one sentence on notable lease terms). The visualization: a large horizontal bar chart, one bar per project, sorted descending by privatePct, each bar animating its width in on load with an eased transition, with the private share labelled at the end of each bar and the project name on the left. Render the Portland Moda Center bar in the ruby-red accent and pin it with a small label reading PROPOSED so it stands out even though several other public-heavy deals also sit near zero. Clicking any bar opens a detail panel showing total cost, the public versus private split, the annual rent note, and the lease terms. Include a sort toggle to switch the ranking between private share and total project cost, animating the bars to their new positions. Add a short, honest caption noting that several of these public-heavy deals still secured real rent, revenue sharing, or relocation penalties that the Portland proposal does not, so the bar chart understates how far Portland sits from the norm. Build the chart by hand with SVG; do not use any external charting library. STYLE — Civic public-ledger look: ivory paper background, near-black ink, a muted grey for secondary text, and ONE ruby-red accent (#DA291C) reserved for emphasis and current-state markers (the Portland row, the break-even line, the running total). Tabular numbers in a monospace font. Document-like, calm, generous whitespace, thin hairline rules between sections, no clutter. The charts are the hero of the page: make them large, smooth, and clearly labelled with axes, ticks and a legend. Animate on load and on interaction with eased transitions. Fully responsive; tooltips on hover. Single-file React using useFireproof for persistence. Keep all copy factual and dryly wry — never hype, never editorialize beyond a light raised eyebrow. No emojis anywhere.
